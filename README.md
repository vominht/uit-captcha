# Auto Fill CAPTCHA UIT

Repository này chứa userscript dành cho **iOS** và **Windows** dùng để tự động điền captcha trên các website của UIT:

- https://student.uit.edu.vn  
- https://daa.uit.edu.vn  

---

## Cách cài đặt (Windows)

1. Cài extension **Tampermonkey** trên trình duyệt.
2. Truy cập đường link sau:  
   https://github.com/vominht/uit-captcha/raw/refs/heads/main/uit-captcha.user.js
3. Nhấn **Install** để cài đặt và bắt đầu sử dụng.

---

## Cách cài đặt (iOS)

1. Tải app **Userscripts** trên App Store:  
   https://apps.apple.com/us/app/userscripts/id1463298887

2. Mở app **Userscripts** và thiết lập **Directory** theo hướng dẫn trong app.

3. Vào **Cài đặt** trên iOS:
   - Chọn **Safari**
   - Chọn **Phần mở rộng**
   - Bật **Userscripts**
   - Chọn **Cho phép trên tất cả website (Allow on All Websites)**

4. Mở **Safari** và truy cập đường link sau:  
   https://github.com/vominht/uit-captcha/raw/refs/heads/main/uit-captcha.user.js

5. Trên Safari, nhấn vào biểu tượng **aA** (bên trái thanh địa chỉ).
   - Chọn **Userscripts**.
   - Một cửa sổ cài đặt userscript sẽ hiện ra.
   - Nhấn **Install** để cài đặt và bắt đầu sử dụng.
